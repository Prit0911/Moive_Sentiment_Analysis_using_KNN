Email Spam Classifier using KNN
This project is a machine learning-based movie sentiment analyzer.
It uses TF-IDF vectorization and the K-Nearest Neighbors (KNN) algorithm 
to classify movies sentiment as 0(sad) or 1 (happy).

Requirements
- Python 3.8+
- pandas
- scikit-learn
- imbalanced-learn
- numpy

This project uses the IMDB Movie Ratings Sentiment Analysis dataset. I didn't upload the dataset because it's size is more than 25MB. 
But you can download it from the kaggle.com and search IMDB Movie Ratings Sentiment Analysis this. You will get that dataset.
It contains 14055 sad sentiments and 13945 happy sentiments.
Columns:
- label : 0/1
- text  : the text message that contains the sentiment
